# Recommended VS Code Extensions

Here is a list of recommended extensions to enhance your development experience involved with this project:

1.  **Live Server** (`ritwickdey.liveserver`)
    *   **Why:** Essential for running your HTML/CSS/JS frontend locally with hot-reloading.
2.  **Prettier - Code formatter** (`esbenp.prettier-vscode`)
    *   **Why:** Keeps your code consistently formatted (HTML, CSS, JS).
3.  **ESLint** (`dbaeumer.vscode-eslint`)
    *   **Why:** Catches JavaScript errors and potential bugs early.
4.  **DotENV** (`mikestead.dotenv`)
    *   **Why:** syntax highlighting for `.env` files.
5.  **PostgreSQL** (`ckolkman.vscode-postgres` or others)
    *   **Why:** Useful if you want to connect to your Neon DB directly from VS Code.
